## ⚡ Core Application Features

StackVest is packed with premium, high-end components:

### 1. Unified Cockpit Dashboard
* **Metrics Tracker**: Live summaries of Net Worth (Cash + Assets equity value), Liquid Capital, Portfolio Valuations, and Dividend cash flow.
* **Custom Bezier SVG Growth Chart**: A responsive line chart rendering historical net worth trends, with linear color gradients and interactive data node values.
* **Custom Risk Allocation Donut Chart**: Visualizes portfolio diversification across Low, Medium, and High-Risk stack assets using custom SVG circular dashes.
* **Seed Capital Injector**: Instantly inject mock VC funding (from $10,000 up to $250,000) directly into the wallet to participate in major auctions.

### 2. Stacks Auction Arena (Bidding Suite)
* **Live Auction Listings**: Browse technology stacks up for auction (e.g. *The Hackathon Holy Grail*, *Vue-Rails FinTech Elite*).
* **AI Competitor Bids**: Built-in backend `MockBidderService` simulates counter-bids from rival venture capital firms and whale investors in real-time, bringing the thrill of the hackathon auction block right to your screen!
* **Detailed Evaluation Panel**: Review full descriptions, tech-stack tags, expected monthly ROI, and complete chronological bidding history logs.
* **Buy Outright**: Lock in immediately and purchase active stacks outright using liquid wallet cash.

### 3. My Portfolio & Yield Payouts
* **Compounding Earnings**: Lists all owned stack assets alongside cumulative yields generated.
* **"Claim Portfolio Yields" Portal**: Manually trigger payout calculators to credit the wallet with outstanding compounding yields generated by owned stacks!
* **Liquidation Block**: Sell any owned stack back to the market at its current valuation. Cash is immediately credited back to the wallet, and the stack is returned to the active Auction Arena block.

### 4. Categorized Audit Ledger
* **Full Logs**: Transparency of all monetary movements (VC deposits, stack purchases, liquidations, and yield dividends).
* **Filters**: Audit accounts easily using interactive filters (Deposits, Purchases, Dividends, Liquidations).

## 🚀 Setup & Launch Instructions

### Prerequisites
Make sure you have **Node.js** (v20+ recommended) and **Ruby** (v3.2+ with Rails 8) installed on your system.

### 1. Boot up in Development Mode (Concurrently)
From the root workspace directory, run the unified launch command:
```powershell
npm run dev
```
> [!NOTE]
> This command will boot up both servers simultaneously:
> * **Rails Backend Server**: Running on `http://localhost:3000`
> * **Vite Frontend Server**: Running on `http://localhost:5173`
>
> Simply open your browser and navigate to **`http://localhost:5173/`** to interact with **StackVest**!

### 2. Single-Server Production Mode (Fully Compiled)
To build and run the application as a single hosted Rails server (serving compiled frontend assets from `/public`):
1. Compile Vue frontend:
   ```powershell
   npm run build:frontend
   ```
2. Start standard Rails server:
   ```powershell
   rails server
   ```
3. Open **`http://localhost:3000/`** in your browser!
